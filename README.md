# 课程简介

本课程主要讲解了如何使用WPF开发经典《贪吃蛇》游戏。

本课程以经典游戏《贪吃蛇》为例，通过综合C#与WPF的编程中的多种技巧，从而整合成一个完整的游戏项目。赵老师在每一段关键代码的实现中，都进行了详细的讲述，力求新手都可以看明白。学完该课程后，若再加以练习，基本上可以套用此编程模式用WPF实现市面上大多数小游戏。
